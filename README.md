# PointPath

**Maximize your credit card points by finding the best transfer partner award flights**

## Overview

PointPath helps travelers get 2-5x more value from their credit card points by comparing transfer partner award availability against travel portal bookings. Instead of using points at fixed 1.5¢ per point through Chase Travel Portal, discover when transferring to airline partners like United MileagePlus offers 2.8¢+ per point value.
